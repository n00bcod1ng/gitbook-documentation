# Knowledge Base

After resolving the initial ticket queue, the most repeatable issues were converted into knowledge base articles — mirroring how real service desks reduce repeat ticket volume and enable faster resolutions. Articles follow the industry-standard ServiceNow KB structure: Symptoms, Cause, Resolution, and Escalation guidance.

Two article types are included, each written for a different audience:

* **Technician articles** (KB-001, KB-002) — internal documentation with diagnostic steps, escalation paths, and prevention guidance
* **End-user article** (KB-003) — a plain-language self-service guide, written so employees can resolve common issues without submitting a ticket
